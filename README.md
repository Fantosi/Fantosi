# Fantosi
### Tell me your Fantosi
블록체인 기술을 활용하여 투명하게 팬덤의 자금을 보관하고, 팬들의 의견을 민주적인 방식으로 수집하여, 모인 자금을 운용함으로써 아티스트와 팬덤의 영향력을 확장시킬 수 있습니다.
